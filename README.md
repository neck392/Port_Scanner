# Port_Scanner
### Check open ports
There are 2 ways to check open ports
<ul>
  <li>PortScan.py</li>
  <li>PortScan.ps1</li>
</ul>

## How to run
```
git clone https://github.com/neck392/Port_Scanner.git
cd Port_Scanner\dist
PortScan.exe
```
Executable file path : .\dist\PortScan.exe
